The following table indicates which versions of Para Mobile currently receive security updates.

| Version | Supported |
|---------|-----------|
| 1.x.x   | Yes       |
| < 1.0.0 | No        |

---

## Reporting a Vulnerability

If you discover a security vulnerability in Para Mobile, please report it responsibly by contacting the maintainers directly via email.

**Do not open public GitHub issues for security vulnerabilities.**

### Reporting Procedure

1. Send an email to the project maintainers describing the vulnerability.
2. Include steps to reproduce the issue, potential impact, and any relevant technical details.
3. Allow reasonable time for the maintainers to investigate and address the issue before public disclosure.

### Response

The maintainers will acknowledge receipt of your report and provide updates on the remediation timeline. Upon resolution, appropriate credit will be given to the reporter if requested.
